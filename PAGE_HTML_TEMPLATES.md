# Dynamic Page HTML Templates

Use these snippets in the admin page editor after creating pages with these exact slugs:

- `privacy`
- `terms`
- `contact`

Set the contact email from Admin > Visual identity > Contact email. The contact page uses `{{contactEmail}}` and the site replaces it from the database.

## privacy

```html
<h2>سياسة الخصوصية</h2>
<p>نحن نحترم خصوصية زوار الموقع ونتعامل مع البيانات بالحد الأدنى اللازم لتشغيل الخدمة وتحسينها.</p>

<h3>المعلومات التي نجمعها</h3>
<p>قد نجمع بيانات استخدام عامة مثل عدد الزيارات ونوع الأداة المستخدمة، دون استخدامها للتعرف الشخصي على الزائر.</p>

<h3>ملفات الارتباط والإعلانات</h3>
<p>قد تستخدم بعض خدمات الإعلانات أو التحليلات ملفات ارتباط لتحسين تجربة العرض وقياس الأداء وفق سياساتها.</p>

<h3>حماية البيانات</h3>
<p>نستخدم إعدادات وأدوات آمنة قدر الإمكان، ولا نبيع بيانات الزوار أو نشاركها خارج الحاجة التشغيلية للموقع.</p>

<h3>التحديثات</h3>
<p>قد يتم تحديث سياسة الخصوصية عند تطوير الموقع أو إضافة خدمات جديدة.</p>
```

## terms

```html
<h2>الشروط والأحكام</h2>
<p>باستخدامك لهذا الموقع فإنك توافق على هذه الشروط وعلى استخدام الأدوات كما هي لأغراض معرفية وتنظيمية.</p>

<h3>استخدام الأدوات</h3>
<p>نتائج أدوات التاريخ والحسابات الزمنية مقدمة للمساعدة فقط، ويجب مراجعة الجهات الرسمية عند الحاجة إلى اعتماد قانوني أو رسمي.</p>

<h3>الاستخدام المقبول</h3>
<p>يمنع استخدام الموقع بطريقة تضر بالخدمة أو تحاول تجاوز الحماية أو استهلاك الموارد بشكل غير طبيعي.</p>

<h3>الإعلانات والروابط</h3>
<p>قد يحتوي الموقع على روابط أو إعلانات خارجية، ولسنا مسؤولين عن محتوى المواقع الخارجية أو سياساتها.</p>

<h3>تعديل الشروط</h3>
<p>قد تتغير هذه الشروط لاحقا، ويعد استمرار استخدام الموقع موافقة على النسخة المحدثة.</p>
```

## contact

```html
<h2>اتصل بنا</h2>
<p>يسعدنا استقبال ملاحظاتك واقتراحاتك أو أي مشكلة تقنية تواجهك أثناء استخدام الموقع.</p>

<h3>البريد الإلكتروني</h3>
<p>
  يمكنك التواصل معنا عبر:
  <a href="mailto:{{contactEmail}}">{{contactEmail}}</a>
</p>

<h3>الدعم</h3>
<p>للطلبات التي تحتاج متابعة منظمة، استخدم صفحة الدعم حتى يتم تسجيل الطلب ومراجعته.</p>
```
